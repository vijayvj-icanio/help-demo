19.0.3 Image List

Images identified by TVT and other images that have been defined as such, by using the Image Mode, are shown in the Image List as Image Deletions and Image Insertions. Similar images are statistically evaluated and automatically linked into Image Pairs by TVT during comparison. Images can also be manually linked.

19.0.3.1 Image List Actions

Reviewing and editing the Image List can be achieved in three different ways:

  • Context (Right-Click) Menu

    • Most of the menu items for reviewing and editing of the images are similar to the ones used for text deviations.

    Additional or adjusted menu items are the following:

    • Pick for Linking: Picks an Image Deletion or an Image Insertion for linking with its counterpart.
    • Link with Picked: Links the selected Image Deletion or Insertion with the already picked for linking one.
    • Split Pair: Splits an Image Pair into an Image Deletion and an Image Insertion.
    • Image Comparison View: Opens the Image Comparison View where Image Pairs can be further inspected.
    • Suggest Matching Image: Gives hints for the counterpart of the selected Image Insertion or Image Deletion. Three suggestions are made for best matching images. When selecting one of the options, TVT navigates you to the best, second or third best match on the other document side. As described in chapter 41.3.3 "Image Mode Button" on page 1, the context menu of an Image Area also helps when searching for the counterpart image.
    • Delete: Deletes an Image Pair, Image Insertion or Image Deletion.
  • Project Window Toolbar

  • Shortcut Keys
    The shortcut keys for reviewing and editing images correspond with the shortcuts for the deviations described in chapter 34.6.1.3 "Deviation List Actions" on page 1. In addition, “i” can be used to open the Image Comparison Details View of an Image Pair when it is selected and Escape can be used to close this view.

19.0.3.2 Image Comparison View Window

Underneath the selected Image Pair in the middle of the Image Comparison View window, there are four different options to display it:

  • Wipe: Both images are overlapped. You can drag the separator or the slider thumb in the slider control to the left and to the right in order to see more of the original or copy image area.
  • Blend: Both images are blended together and initially 50% of each image is displayed. You can adjust the percentage of the pixel value of each image by moving the slider thumb to the left (original pixel value increases) or to the right (copy pixel value increases).
  • Blink: Both images are superposed and shown alternately in a blinking motion. The color code of the document side of the currently displayed image (either blue or red) will also be blinking for a better orientation. You can change the blinking speed by moving the slider thumb to the left (faster) or to the right (slower).
  • Difference: The difference that is calculated over the pixel value of both images is graphically displayed. If the difference is zero, there will be no deviation displayed and the color of the image will be identical with the background color.

Next to the four viewing options, there is another tab control for zooming and rotating the view:

The “Whole View” tab controls the zoom level and rotation of the Image Pair synchronously. Select a rotation degree from the rotation drop-down list or clicking on the rotation buttons rotates the view in 90° increments. The zoom drop-down list and the plus and minus buttons let you zoom into and out of the view. You can also type a zoom percentage directly into the zoom field. The Reset button restores the view to 0° rotation and 100% zoom.

It is also possible to zoom with the mouse: Hold the Ctrl key and use the mouse wheel.

An arrow to the left in the zoom field indicates whether the copy image is currently scaled up or down relative to the original image. An upward arrow indicates up-scaling. For example, “↑200%” means that the copy image is scaled by a factor of two horizontally and vertically vs. the original image. A downward arrow indicates down-scaling; e.g. at “↓200%” the original image is displayed at twice the copy image’s dimensions.

The four arrow buttons shift the copy image by one pixel relatively to the original image. Note that the meaning of “one pixel” depends on how far you are zoomed into the view. If you need more fine-grained control over the alignment of the images, switch to the “Whole View” tab and select a higher zoom level, then return to the Adjust Copy Image tab and use the arrow buttons on the zoomed images.

 

It is also possible to adjust the copy image with the mouse while the “Adjust Copy Image” tab is open: To move it, press the left mouse button over the image, drag and drop it in the required position. To resize the copy image, go over it with the mouse cursor and then move the mouse wheel back (the image becomes smaller) or forth (the image gets bigger). To rotate the copy image counterclockwise, right-click on it. To rotate it clockwise, press the Ctrl key and right-click simultaneously. To rotate the image by 180°, press the Shift key and right-click simultaneously. Please note that after each move or resize action, a preview is initially shown and a full quality rendering appears shortly after. The following shortcut combination can help you:

19.0.3.3 TVT Artwork: Image Comparison View Window

If the optional TVT Artwork module is activated (please see chapter23.7.8 "About" on page 1), the Image Comparison View is divided into three panes. In the middle, the selected Image Pair is displayed. The left pane displays an overview of the Original document and the selected image is framed in green. The right pane displays an overview of the Copy document and the selected image is also framed in green. The size of the left and right overview panes can be adjusted by dragging at their inner edges.

At the bottom left of the Image Comparison View window there are two additional buttons: "Detect deviations automatically" and "Mark deviations manually".

Detect Deviations Automatically

By pressing this button, you ask TVT Artwork to automatically detect deviations in the current Image Pair. It will try to recognize differences between the images by means of detecting differences in both luminance and pixel chrominance, meaning both grayscale and color differences can be detected. Detected deviations within the Image Pair will be included as Image Pair Children of an existing Image Pair, thus converting the Image Pair into an Image Pair Parent.

Mark Deviations Manually

Activate this button to manually mark deviations in the current Image Pair. TVT Artwork allows you to individually inspect and mark a certain area of an image as a deviation and manually create an Image Pair Child. With the "Mark deviations manually" button activated and the mouse button pressed, drag to draw a box on the image. When you release the mouse button, the new Image Pair Child gets a green dashed outline and is added at the bottom of the Image List.

An Image Pair containing Child Images is considered an Image Pair Parent. When an Image Pair Parent is marked, all Image Pair Children that are part of it, are automatically marked accordingly. The state of the Image Pair Parent also depends on whether the individual Image Pair Children are accepted or marked.

Please note: Image Pair Parents are automatically locked. In order to unlock or split these Image Pairs, all Image Pair Children have to be detached or deleted first, by selecting the preferred option in the pop-up warning message. In the Image Comparison View, “Adjust Copy Image” can only take place, if the Image Pair Children are deleted first.

When zooming is applied in the Image Comparison View, a green outline indicating the zoomed in area appears within the green dashed outline of the image, on both left and right panes.
