40.15 Image Areas

40.15.1 Image Area Import from Documents

The option buttons “Never detect images”, “Ask before detecting images” and “Always detect images” allow you to set the TVT behavior when loading documents. By default, TVT is set to “Ask before detecting images” in your documents.

 

“Ask before detecting images” means that an “Image Area Import” message is going to appear each time you are creating a TVT project.

This default behavior can also be influenced if you check the “Don’t show this message again” checkbox before deciding to detect or skip the detection of the images. If you chose “Detect Images”, the default behavior will automatically be set to “Always detect images”. If you chose “Skip Images”, the default behavior will automatically be set to “Never detect images”.

The “Let me decide for each document” option allows you to detect or skip the detection of images selectively for each document on the Original and Copy side. Closing the “Image Area Import” message without making a decision, triggers the “Let me decide for each document” option as well.

40.15.2 Minimum Dimensions for Image Area Import

The minimum height or width of the imported image areas can be set with this option.

40.15.3 PDF Options

The following detection methods can be selected from the drop-down menu:

Visual + Bitmap (default): Detects images using a visual page analysis as well as pixel images.

Box + Bitmap: Detects images surrounded by boxes using a geometric shapes analysis as well as pixel images.

Bitmap: Detects pixel images.

Visual: Detects images using a visual page analysis.

Box: Detects images surrounded with boxes using a geometric shapes analysis.

Named Color: Given that geometric shapes with certain color names are placed behind vector graphics, the color names of these shapes are assessed for the detection of images.

Special Color: A special case of Named Color - all the geometric shapes that have a named color are detected as images.

By default, the “Combine Image Areas” checkbox is checked, meaning that overlapping image areas are handled as one image area.

40.15.4 Automatic Linking

The following options for processing identical images can be selected from the drop-down menu:

Link (default): Displays the linked images in the Images List for processing.

Link & Accept: Displays the linked images in the Images List as automatically accepted.

Remove: The Image Pairs do not appear in the Images List at all.
