30.5.0.1 Style Transition Details

To display all instances of a particular style transition, simply select it. The instances will be listed in a table in the Details Area to the right of the Style Transitions List. This table consists of a checkbox column, a “Text” column and an “Override Comment” column. You can navigate to the exact location in the text of each instance, by clicking on it in the table. It is also possible to navigate from a particular mapped word in the document to the style transition instance in the Style Transition Details table. Simply double-click on the word in the document and it highlights that instance.

If a comment has already been added to a style transition in the Style Transitions table, the text of the comment will not be visible in the Style Transitions Details table, but the checkbox will be checked and the whole line will be highlighted in red.

Overriding the comments can be achieved in two ways. After highlighting the line containing the transition in question, you can either:

Press Enter or double-click the “Comment” column. This opens the override comment box for manual input of text through the keyboard.

+
Pastes the original style into the comment box.
+
Pastes the copy style into the comment box.

Saves the text and closes the comment box.
Closes the comment box without making changes to the comment.

You can also use the following shortcut keys to enter an automatic comment or to reset the comment box of the Style Transition.

Checks the checkbox.

This sets the selected transition to “unacceptable” by highlighting the line in red and entering an automatic comment into the override comment box in the form “Style is x in original and y in copy”.
Resets the individual transition to that of the group from which the details are drawn.

Unchecks the checkbox.

This sets the selected transition to “acceptable” and removes any comments from the comment box.

Altering a comment will not affect the status of the checkbox and the line will remain marked as unacceptable (highlighted in red) in the table. Removing the comment will reset the status and enter a check mark in the checkbox and remove the red highlighting.
