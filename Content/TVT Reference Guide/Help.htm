23.7 Help

23.7.1 View Help

This option opens the Text Verification Tool Help. This gives you access to three help files: the User Manual, the Reference Guide and the Special Topics.

23.7.2 User Manual

This option opens the User Manual. It provides an introduction to TVT and describes how to complete a project in 6 Easy Steps.

23.7.3 Reference Guide

This option opens the present document.

23.7.4 Special Topics

This option opens the Special Topics document. This document contains more in-depth information about features and components of TVT.

23.7.5 Legend

+

This option opens the color legend in the Details Area. It is a key to the colors used in TVT, e.g. to identify different types of deviations. For further information, please see chapter 40.2.5 "Legend Area" on page 1.

23.7.6 E-Learning

Clicking on this item takes you to the TVT E-Learning Platform, where you can directly log in with your TVT E-Learning credentials. If you are not a registered E-Learning user, you can find out more about TVT E-Learning on the platform and request a free trial.

23.7.7 Contact Support

Clicking on 'Contact Support' opens a link in your browser. You are prompted to enter and submit information about your case.

23.7.8 About

Important copyright information is displayed in the “About TVT” window. The following items are listed as well:

  • Version: The TVT version you are using
  • Computer Code: The computer code of your TVT installation
  • Unlock ID: The license unlock number used for internal purposes
  • Serial #: Your assigned TVT serial number
  • Expires: The support expiration date of your license
  • Activated Modules: TVT Artwork, TVT Spelling and TVT Barcode are optional modules available upon request. In addition, TVT Spelling can be acquired together with the Stedman’s Medical Dictionary. Modules that are available to you will appear in color and the inactive ones are grayed out. Clicking on the icon of a module takes you to our product website www.schlafenderhase.com. For more information on the modules, contact support@sh-p.com.
