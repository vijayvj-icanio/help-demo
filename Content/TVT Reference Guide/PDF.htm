9.7 PDF

9.7.1 Import

9.7.1.1 Margin Exclusion Areas:

Pages of PDF documents can optionally be annotated with invisible boxes that separate a page’s content from extraneous information outside the printed area.

When any of these options are checked, TVT will automatically create Exclusion Areas covering the outside of such boxes when importing PDF files. For well-prepared PDF files, “Trim Box” should usually be the right choice.

9.7.1.2 Annotations:

The options “Do not load annotations”, “Ask before loading annotations” and “Automatically load annotations” are available from the drop-down and allow you to set the behavior of TVT when opening PDF documents containing annotations. By default, TVT is set to “Ask before loading annotations”.

“Ask before loading annotations” means that an “Annotation Import” message will appear each time a PDF document containing annotations is loaded.

Imported annotations are always read-only. The annotation types Replace, Strikethrough and Underline Text are also imported and shown by TVT.

By activating “Only load annotations created by TVT”, the options described above will only apply to annotations previously exported using TVT. All annotations created in Adobe Acrobat will be ignored.

9.7.1.3 Corrections:

By default, “Import for comparison” is checked, meaning that Correction Comparison is enabled. For more information, please see 5.5.1.1 "Corrections" on page 1.

“Show layer selection” is checked by default, meaning that during document loading, a “PDF Layer and Spot Color Selection” window will pop up for each PDF file with detected layers.

Similarly, “Show spot color selection” is checked by default and during document loading, a “PDF Layer and Spot Colors Selection” window will pop up for each PDF file with detected spot colors. In addition, certain spot colors can be hidden. When the name of a spot color is entered in the “Hide spot colors” field, elements in this color are not displayed. More than one spot color names can be entered separated by semicolons.

For more information on PDF layers and spot colors, please see See Special Topics "Layer and Spot Color Selection" on page 1 Portable Document Format (PDF) Import.

9.7.1.5 Simulate Overprint:

Overprinting is turned off by default, but can be turned on, if needed.

Alternatively, the overprint can be simulated via the Document Settings button.
