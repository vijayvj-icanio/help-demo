40.12 Reading Order Changes (ROC) or Order Breaks (OB)

Depending on whether the grouping of Order Breaks to Reading Order Changes is enabled or not, either the ROC or the OB indicator will be displayed in the status bar.

This button controls the display of information on reading order changes in the deviation list. It indicates the number of Reading Order Changes (ROC) edited/found (0/6).

If there are any unedited Reading Order Changes, the numbers are in red (0/1). When all Reading Order Changes have been edited, the numbers are in blue (1/1).

Selecting a Reading Order Change in the Deviation List causes the moved text to be selected on both sides and the view of the Details Area to be changed to display the same text, highlighted in green and framed by a thick red line, as well as the text before and after it in both original and copy files.

To expand the view of a Reading Order Change, click the plus control at the beginning of a Reading Order Change entry. All Order Breaks that are part of that Reading Order Change are displayed below it. When a Reading Order Change is marked, all Order Breaks that belong to it also get marked.

Please note that the text of marked Reading Order Changes is framed by a red line in the exported documents as well. In the report, Order Breaks that are part of a Reading Order Change will not be listed by default. You can adapt this behavior in the User Settings. please see chapter 30.5 "Report Creation" on page 1 for more details.

This button controls the display of information on order breaks in the deviation list. It indicates the number of order break deviations (OB) edited/found (0/6). When there are unedited deviations, the numbers are in red (0/6). When all deviations have been edited, the numbers are in blue (6/6).

Clicking an Order Break selects the word, and the reading order before/after it is graphically displayed in both documents. A blue arrow indicates the reading order of the original and a red arrow indicates the reading order of the copy.

For more information on Order Breaks, please see See Special Topics "Comprehending Order Breaks" on page 1 Comprehending Order Breaks.
