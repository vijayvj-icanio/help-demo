16.1 TVT Barcode

16.1.1 Detect Barcodes

For detecting barcodes, you can choose one of the following options:

  • First and last page of document
  • All pages
    • Please note: Detecting barcodes on all pages may slow down the loading of your documents
  • Ask before detecting barcodes
  • Do not detect barcodes

TVT Barcode is set to detect barcodes on "First and last page of document" by default. If set to "Ask before detecting barcodes", each time you are creating a project, a "Barcode Import" message will pop up during document loading, prompting you to decide whether and where to detect barcodes. If you first tick the "Don't show this message again" checkbox at the bottom of the pop-up window, the chosen behavior is set for further projects. Whenever needed, you can go back to the TVT Barcode tab in the User Settings to set another behavior.

The option "Grade and display results" is checked by default. After decoding the barcodes, TVT Barcode will access the grading parameters of each 1D barcode. For more information on barcode grading, please check 5.5.10 "Processing Barcodes" on page 1.

16.1.2 Auto-Detected Barcodes

The barcode types that can be detected by TVT Barcode are listed and many are selected by default. If a barcode isn't selected in the list, TVT Barcode will not detect it. You can decide which barcodes should be automatically detected by making sure they have a checkmark placed next to them. For more information and a full list of the barcodes that can be detected by TVT Barcode, please see See Special Topics "Barcode Decoding" on page 1 Barcode Decoding.
