34.6.1 Deviation List

 

34.6.1.1 Deviation List Structure

The Deviation List includes the following information:

  • Type: Each deviation type is represented by a different icon (please see chapter 34.6.1.2 "Types of Deviations" on page 1” or See Special Topics "Deviation Types" on page 1 Deviation Types; for information on warnings, please see chapter 16.0.5 "Warning Settings …" on page 1.).
  • Original Text: Text as it appears in the original document.
  • P.: Page number of the original text.
  • Copy Text: Text as it appears in the copy document.
  • P.: Page number of the copy text.
  • Comment: This field displays an icon indicating how the deviation has been processed, e.g. marked as an error in the copy, accepted, etc. (please see chapter 34.6.1.3 "Deviation List Actions" on page 1 for a description of the processing options) and the accompanying comment or instruction.

TVT allows you to sort and filter results from the Deviation List columns. For more details, please see the 5.4.2 "Sort and Filter in the Deviation List" on page 1

34.6.1.2 Types of Deviations

Deviation: This indicates a text deviation – text that is different in the original and the copy.

Insertion: This indicates text that is not present in the original but is found in the copy.
Deletion: This indicates text that is present in the original but not found in the copy.

Warnings: These are mapped deviation pairs where the deviating characters are compatible according to the Unicode Standard. For more detailed information on warnings, please see the See Special Topics "Warnings" on page 1 Warnings.

34.6.1.3 Deviation List Actions

Reviewing and editing the Deviation List can be achieved in three different ways:

  • Context (Right-Click) Menu
  • Project Window Toolbar
  • Shortcut Keys

The following actions can be performed:

Standard Window Keys

The Up Arrow key takes you to the previous deviation in the list.

The Down Arrow key takes you to the next deviation in the list.

The Home key takes you to the beginning of the list.

The End key takes you to the end of the list.

The Page Up key takes you to the top of the current deviation list view.

The Page Down key takes to you the bottom of the current deviation list view.
