36.14.1 Open…

+

Invokes the Open file dialog, allowing you to select and load an existing project.

Projects created using TVT Version 12.0 have the extension “.tvp12”. Projects from earlier versions (with extensions such as “.tvp11”, “.tvp10”, ...) cannot be imported into TVT 12.0; only project files with the extension “.tvp12” can be opened with TVT 12.0.

Files compared in a saved project must be accessed through the same file paths that were used when the project was first created.

If a project file has been moved or renamed, TVT will notify you that one or more files cannot be located and you will need to locate the new file path. Click on “OK” to open the Open file dialog, which is preset with the required file name and extension, e.g. DOC or PDF.

If you click on “Cancel”, a message will notify you that the project contains an invalid path, and the project will be opened without the file.

If a modified version of the document is selected, you will be warned that the document has been altered and will have the option of returning to the file selection dialog.

To avoid this, we suggest that you store the compared documents and the project in the same folder and always keep them together. The paths from the project file to the document files are stored in relation to each other so that if the path relationship between them remains the same, these warnings will not appear.

Because documents can be modified after saving a project, TVT validates the document’s integrity each time a project is loaded.

If a document was modified, a warning will appear and prompt you to select the document compared initially .
