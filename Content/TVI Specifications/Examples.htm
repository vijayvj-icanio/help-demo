16 liana 3. Examples

Detailed examples can be found in the SDK folder under “Samples”. The folder structure is given as:

Carton Sample documents, exclusion template and exclusion word list.
DeviEx Sample documents and critical word list.
Multieast Sample documents and exclusion template.
Schema Common XSD Schema for TVI files. The schema is designed to be syntax checking only.
temp_out Temporary out folder: TVT will store any data generated by sample runs here. The contents of this folder should be deleted after each sample run.
