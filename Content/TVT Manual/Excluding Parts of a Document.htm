5.2.1 Excluding Parts of a Document from Comparison

If one of the documents contains text that is not intended for printing or shouldn’t be compared, it can be marked for exclusion prior to running the comparison. TVT offers various options for excluding content:

 

  1. Exclusion Area Mode

    To exclude text from the comparison, click on the Exclusion Area Mode button in the toolbar. Then draw a frame around the content to be excluded by clicking and dragging. The size of the placed area can then be increased or reduced manually by dragging on its edges.

    An Exclusion Area can be deleted by right-clicking on it and selecting “Delete” from the context menu. This function is only available when the Exclusion Area Mode is active. If the text you are excluding is repeated in exactly the same position on subsequent pages, the Exclusion Area can be repeated on those pages automatically. To do so, right-click on the Exclusion Area and select “Show on all pages” from the context menu.

    Furthermore, if the text is repeated on odd, even or particular pages, select "Show on Specific Pages..." from the right-click menu. You can choose a repetitive option or enter the page numbers for the repetition of the current exclusion area.

    All repetitive Exclusion Areas are linked and altering one on one page will have the same effect on all pages. Exclusion Areas that have been set before starting the comparison cannot be modified afterwards. You have to reset the project to modify them. If you reset your project (please see 23.13.3 Reset Comparison C:\TFS\Docs\FlareDoku\TVT Documentation\Output\Yuliana_Wong\TVT Web New Target\Content\TVT Reference Guide\Reset Comparison.htm file:///C:/TFS/Docs/FlareDoku/TVT Documentation/Output/Yuliana_Wong/TVT Web New Target/Content/TVT Reference Guide/Reset Comparison.htm Reset Comparison 1 1 on page 1), it returns to the status it had prior to the comparison process. All your deviation comments, for example, will be lost. However, a pop-up window gives you the option of retaining the Exclusion Areas that have been set before comparing the documents. It is possible to edit these areas prior to carrying out a new comparison.

     

    Please note: For documents that have the same layout and thus, the same areas to be excluded every time, you can create an Exclusion Area Template. Using the Exclusion Area Mode, mark the areas that have to be excluded on one page of the document, and select “Save Exclusion Template” from the drop-down list of the Document Settings button in the Document Window Toolbar. You have to define the repetition frequency of the template (on each page, on every second page etc.). Once saved, this Exclusion Area Template can be loaded by selecting “Load Exclusion Template” from the Document Settings drop-down list . For further instructions, please see See Special Topics "How to Use Exclusion Area Templates" on page 1 How to Use Exclusion Area Templates

     

  2. Inclusion Mode

    To include a specific text section in the comparison, and that is, to exclude the rest of the text, activate the Inclusion Mode under the Document Settings before comparison. All text gets highlighted in blue and is excluded. At the same time, the Exclusion Area Mode button is activated and its functionality, explained above, is now inverted. Instead of excluding text, you can draw areas to include text. In case you have placed Exclusion Areas before activating the Inclusion Mode, these areas will be converted and the text within the areas will be included for comparison. Please see See Special Topics "How to Use the Inclusion Mode" on page 1 How to Use the Inclusion Mode

     

  3. Document Divider Mode

    The Document Divider Mode is especially useful if you wish to exclude large areas of a document, e.g. several pages. This mode allows you to place a divider across the entire width of the document; it is therefore not suitable for excluding text in columns.

    Document Dividers can only be applied prior to pressing the “Compare” button. At this point, all of the text is highlighted in orange because the text has been mapped but not yet divided. Document Dividers can be placed by double clicking into the original document or into the copy where an exclusion area or inclusion area is to start. The first double click marks the beginning of the area to be excluded, the second marks the end of the excluded area. Once the document has been divided into included and excluded areas, the parts that will be included in the comparison remain highlighted in orange, while those to be excluded are highlighted in blue.

     

  4. Exclusion Word List Function

    Exclusion Word Lists help to facilitate the exclusion of predefined content, for example, the headers from the QRD templates when comparing labels.

    Exclusion Word Lists are available for the original as well as the copy document. The default Exclusion Word List is “Simulated Lines” for both.

    A different list can be selected per document side prior to comparison. To do so, select the “Exclusion Word List” option from the drop-down menu of the Document Settings button . In the Exclusion Word List drop-down, click on another list. TVT asks you to confirm your choice and after confirming with "Yes", the selected list will be applied.

    To set a different default Exclusion Word List or no default Exclusion Word List at all, you have to adjust the User Settings. The word count in the status bar will reflect any of these actions. If the word count is similar when using the Leaflet or the Document settings, the project is ready for the comparison. The report will contain information on the placed Exclusion Areas, Document Dividers and the selected Exclusion Word List.
