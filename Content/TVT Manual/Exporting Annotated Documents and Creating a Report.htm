5.6 Exporting Annotated Documents
and Creating a Report

When you are finished checking the files, you can create a PDF containing all markings, comments and annotations, as you have them in your TVT project. From the “Project” menu, select “Export Copy as Annotated PDF”. You will be asked if you want to open the document once it has been saved and where to save it. TVT will suggest a new file name with some appended text ending “_ev” (exported version) added to the file name, in order to distinguish it from the original file name of the compared document.

If any deviations have been marked and annotated in the original document, it can also be exported as a PDF in the same manner. From the “Project” menu, select “Export Original as Annotated PDF” and proceed as above.

The exported documents can be opened and viewed by using a PDF reader, preferably an Adobe Acrobat Reader. All annotations in the exported files are locked, meaning that they cannot be deleted, edited or moved.

At the upper left corner, each exported file has an automatically created comment with information on the project and the person who has created it.

To create a report of your project, select “Create Report” from the “Project” menu. TVT can generate the report in PDF, HTML or XML format. The report provides comprehensive information about the project, including names of the compared documents, settings that were used, and project history. It contains the Windows user name(s) of the person(s) who has/have worked on the project, details of identified deviations and how they were handled. If there are any unedited items remaining, warnings will appear at the end of the report. The report also has a space for signature, to confirm that TVT was used to compare the documents and that the recommended steps were performed to ensure the quality of the results. If you do not wish to print the report, you can add a digital signature to the file and save it. The information contained in the report can also be tailored to your own needs. Under the Project menu, select “User Settings” to open the User Setting window. Then select the “Report Creation” tab and bear a check mark to the entries you want included in your reports and remove it from the ones that you do not want to be included in them.
