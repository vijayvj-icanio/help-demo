5.5 Checking Deviations

After performing the comparison, some of the words and areas in the document windows are marked in different ways and can be categorized as follows:

 

  • Text Deviations (unmatched text, unmapped text, map warnings, clones)
  • Reading Order Changes and/or Order Breaks
  • Hyphenations
  • Critical Words
  • Spelling
  • Images

 

The different color markings are described in the Color Legend (please see chapter 8 "Color Legend" on page 1).
