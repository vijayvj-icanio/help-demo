5.5.1 Text Deviations (DEV)

You can toggle the highlighting of text deviations on and off with the “DEV” button in the status bar. Text deviations are differences in the text of the compared documents. These differences can be:

 

Unmapped Text

This type of deviation is marked in yellow and refers to words that are found in both documents but in different contexts or positions.

 

Unmatched Text

This type of deviation is marked in orange and occurs when text is found in one document but not in the other. The text cannot be mapped because it is not found in the compared document. The copy may include text that was not present in the original manuscript or a change may have occurred in the sequence of characters, e.g. a word may be misspelled in the copy so that it is not recognized as a match in the original document.

 

Warnings

Warnings refer to predefined groups of characters that may or may not be acceptable and therefore must be checked by the user. For instance, warnings may appear for spacing, quotation marks and bullets. The groups of characters indicated in a warning may be tolerable. Nevertheless, TVT marks these deviations as Warnings. If characters indicated in a warning are tolerable, they can be accepted by unchecking their checkbox in the Warning Settings. The Warning Settings window can be activated from the Project menu drop-down list or by selecting the Check Warnings Settings option from the drop-down list of the Check tab in the Wizard Toolbar (please see 25 Warnings C:\TFS\Docs\FlareDoku\TVT Documentation\Output\Yuliana_Wong\TVT Web New Target\Content\TVT Reference Guide\Warnings.htm file:///C:/TFS/Docs/FlareDoku/TVT Documentation/Output/Yuliana_Wong/TVT Web New Target/Content/TVT Reference Guide/Warnings.htm Warnings 1 1 on page 1).

 

Clones

If the copy contains multiple instances of the same text and it is present in the original document, the duplicate text areas are called Clones. A clone is the exact copy of an already mapped text.

Clones are only supported under the Cartons project setting (please see See Special Topics "Cartons" on page 1 Cartons). Clones are only checked in the copy and not in the original document. They are highlighted in yellow and framed in blue while Insertions are highlighted in yellow and framed in red.

Only one of the Clone instances is mapped to the original text. The rest of the Clones are its duplicates. When a word in a duplicate Clone is selected, a blue arrow points toward that word in the Clone’s master text. When selecting a Clone in the Deviation List or when double-clicking on the Clone itself, arrows are shown for all words of the Clone.

Upon manual (un-)mapping text, all Clones are recalculated. If a Clone gets disconnected from its master text through unmapping, it is then converted into an Insertion.

 

5.5.1.1 Corrections

The Correction comparison allows you to compare text edit comments of a PDF file (Original) against the already corrected version of the same file (Copy). This comparison implies that:

  • the Original document contains correction instructions;
  • these correction instructions are text edits only, meaning that they are from the PDF comment type: Insertion, Replacement and Deletion
  • the “Corrections: Import for comparison” option is activated in the User Settings (please see 9.7.1.3 Corrections: C:\TFS\Docs\FlareDoku\TVT Documentation\Output\Yuliana_Wong\TVT Web New Target\Content\TVT Reference Guide\PDF.htm file:///C:/TFS/Docs/FlareDoku/TVT Documentation/Output/Yuliana_Wong/TVT Web New Target/Content/TVT Reference Guide/PDF.htm#Correcti PDF 1 1 on page 1).

If you have chosen to import Annotations (please see 9.7.1.2 Annotations: C:\TFS\Docs\FlareDoku\TVT Documentation\Output\Yuliana_Wong\TVT Web New Target\Content\TVT Reference Guide\PDF.htm file:///C:/TFS/Docs/FlareDoku/TVT Documentation/Output/Yuliana_Wong/TVT Web New Target/Content/TVT Reference Guide/PDF.htm#Annotati PDF 1 1 on page 1), after loading the files, the annotation icon will be displayed together with the correction markup for Insertion, Replacement and Deletion accordingly in the Original document. Otherwise, this markup can be toggled on by activating the Old Annotations option from the drop-down list of the Document Settings button .

During comparison, TVT checks whether the correction instructions in the Original have been implemented in the Copy. Depending on the outcome, there can be three different states and thus, three different Correction types:

  • Correction Applied: The correction instruction resulting from the text edit comment has been implemented.
  • Correction Misapplied: The implemented change does not match with the correction instruction.
  • Correction Not Applied: There is no change in the Copy.

By default, after selecting a Correction in the Deviation List, you will see the “Correction vs. Copy” view displayed in the Compare Selections Area to the right.
