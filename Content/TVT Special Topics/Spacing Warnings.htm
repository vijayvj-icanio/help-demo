24.1 Spacing Warnings

Spacing warnings are raised, if a word in one document is divided into two words in the other document, without changing the textual content. Spacing warnings also occur, if a line break or a word hyphenation is explicitly not allowed, but takes place on the other document side. There are Major and Minor Spacing Warnings. Whether a spacing warning is major or minor, depends on the two characters directly adjacent to the space. A spacing warning is considered major if the character to the left of the space has the same type as the character to the right of the space. A spacing warning is considered minor, if the characters have different types.

Character’s Type:

Each Unicode code point is assigned to a “general category” by the Unicode standard; this value is its type for the purpose of spacing analysis. In other words: a spacing warning is major if the two surrounding characters have the same general category, and minor if they have different general categories. Examples of these categories are “letter, uppercase”, “letter, lowercase” and “symbol, other”.

There is an exception to this rule: If the character to the left of a space is either in the “letter, uppercase” or “letter, titlecase” categories, and the one to the right of the space is in the “letter, lowercase” category, this also generates a Major Spacing Warning. Another exception concerns combining character sequences (see 33.1.1 "Combining Character Sequences" on page 1): If the character to the left of a space is part of a combining character sequence, then the general category of that sequence’s base character (i.e. its first character) is considered for the spacing analysis instead.

Major Spacing Warnings are also raised when a non-breaking space is used on one document side, and the rule is broken on the other. There are different kinds of such spacing characters and depending on which one is used, the rule differs. For example, the use of a spacing character can indicate that two items should be kept together and not be separated by a line break; the use of an invisible non-breaking space does not allow the hyphenation of a word at the position it is placed at.

Spacing warnings will be minor, only if the reader has another hint that there should be a space. This hint is taken from the type transition.

Because of the importance of Major Spacing Warnings, hiding them is disabled by most project settings. In the Complex Languages and Special PDFs settings, the Major Spacing Warning checkbox in the Warning Settings window is enabled, in order to allow reduction of irrelevant deviations for certain scripts and broken PDF files.

24.1.1 Asian Spacing Warnings

A special case of Spacing Warnings are the Asian Spacing Warnings. These spacing warnings occur when certain Asian scripts are used, for which it is known that the addition or deletion of a space is allowed. If that is the case, an “Asian Spacing Warning” entry is going to appear in the Warning Settings window as shown on the screenshot below.

By default, the checkbox of the Asian Spacing Warnings is not checked, so they are hidden and not shown in the Deviation List. By checking the checkbox, the Asian Spacing Warnings are made visible in the Deviation List. Note that these warnings can appear independent of the used project setting.
