25.5 How to Use Exclusion Area Templates

Exclusion Area Templates for print information (i.e. all print information outside the printed area) make it possible to reuse Exclusion Areas on documents that share the same layout. They can also be set to repeat after a given number of pages, so that you only have to place Exclusion Areas on the first page or pages of the document.

Follow these steps to create a template:

  1. Create a new project.
  2. Select the document you want to create the template for.
  3. Place Exclusion Areas on all text you want to exclude from the first page.
  4. If your layout is two-sided (like in a book) or has even more different layouts on following pages, place the Exclusion Areas on the next pages, too.
  5. Select “Save Exclusion Template” from the drop-down menu of the Document Settings button . You will be asked to enter the number of pages, after which the template should be repeated. For a single-sided layout, that will be 1, for a two-sided layout 2 and so on. Enter 0 if you do not want the template to be repeated.
  6. Save your template. The “Save as” dialog opens automatically and the name of the template is preset to the name of your document with the Exclusion Area Template extension “_tvpt”.

After saving it once, the template is ready to be used with other documents. You can also load it directly for the active document in order to place the Exclusion Areas on all the following pages. Please note that a template must start on the first page of a document. Title pages, etc. are not taken into account.
