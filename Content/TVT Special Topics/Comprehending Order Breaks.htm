34.9 Comprehending Order Breaks

By default, if there is any difference between the reading order of the compared texts, Reading Order Changes will be detected by TVT. Reading Order Changes consist of Order Breaks and are specially designed to be more user-friendly to view and handle (please see the 5.5.2 "Reading Order Changes (ROC) and Order Breaks (OB)" on page 1). If the grouping is not applicable or not preselected, Order Breaks will be detected instead.

Order Breaks allow you to find shifted text blocks easily. They mark differences in the document flow between two compared documents.

Order Breaks have the following properties:

  • Order Break symbols are always visible, unless the word before or after the Order Break symbol has been selected by the selection tool. In this case the Order Break symbol is hidden to make the characters behind it readable
  • Order Breaks can be located and marked using the deviation list
  • Order Breaks can be marked using the Exclusion and Error Area tools.

There are two different Order Break symbols:

  • A ► marks the start of continuously mapped text block
  • A ◄ marks the end of a continuously mapped text block.

The actual order break occurs somewhere between ◄ and ►. Sometimes there are deviations between the symbols so that TVT cannot locate the exact position of the break.
